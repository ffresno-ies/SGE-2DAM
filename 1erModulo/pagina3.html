Creación del Modelo

Ahora toca crear el modelo. Crearemos el modelo libro que se transformará en una tabla en la base de datos por medio del ORM. Nosotros sólo tendremos que añadir algunos campos adicionales para completar el modelo con los datos de un libro. 

                
from odoo import models, fields

class Libro(models.Model):
    _name = 'libreria.libro'
    _description = 'Modelo para gestionar libros'

    name = fields.Char(string='Título', required=True)
    author = fields.Char(string='Autor')
    isbn = fields.Char(string='ISBN')
    paginas = fields.Integer(string='Número de páginas')
