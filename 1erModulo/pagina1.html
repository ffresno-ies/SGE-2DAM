Usando Scaffold para crear un módulo

El primer módulo que vamos a crear en Odoo es un módulo de gestión de una Librería. Primero entraremos en la terminal del contenedor de Docker donde tenemos Odoo. 

# cd /mnt/extra-addons
# odoo scaffold libreria

En este momento debemos ver en nuestra carpeta de desarrollo local /odoo_dev una nueva carpeta con el módulo libreria. 

libreria/
├── __init__.py
├── __manifest__.py
├── controllers/
│   └── __init__.py
├── models/
│   └── __init__.py
├── views/
│   └── views.xml
└── security/
    └── ir.model.access.csv
