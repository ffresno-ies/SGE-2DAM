Permisos del Modelo

Contiene configuraciones de seguridad, como permisos de acceso para los modelos del módulo. Odoo crea automáticamente un archivo ir.model.access.csv que es un archivo de control de acceso de modelo, donde puedes definir qué grupos de usuarios tienen permisos de lectura, escritura, creación y eliminación en cada modelo.

Para que el modelo libro del módulo libreria pueda ser accedido por todos los usuarios, debemos crear un archivo llamado ir.model.access.csv. Este archivo debe incluir la siguiente información: 

                
id,name,model_id:id,group_id:id,perm_read,perm_write,perm_create,perm_unlink
access_libreria_libro,libreria.libro,model_libreria_libro,base.group_user,1,1,1,1

Recuerda que la línea 'security/ir.model.access.csv', debe estar descomentada. 

                
'data': [
    'security/ir.model.access.csv',
    'views/views.xml',
    ...
],
